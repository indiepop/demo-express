demo-express
============

nodejs,express3,websocket,socket.io,karma,jasmine,demo.

--------------------

这是初学以上一些新东东的一个练习demo，也就是一个聊天室，通过websocket实现其实时性。本实例没有什么jquery，bootstrap等组件，也没有数据库，纯js和css实现，尽量做到简单易懂对学习以上内容起到良好的辅助作用。

通过node app.js就可执行。
